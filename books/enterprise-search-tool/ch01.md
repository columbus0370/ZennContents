# はじめに

フリーランスとして営業をしていると「ターゲット企業を探す → メールを書く」この繰り返しに毎回30〜60分かかっていました。

そこで作ったのが **AI営業アシスタント**です。

- 業種・売上・従業員数で企業を絞り込み（gBizINFO API）
- 選んだ企業に対してClaude APIが営業メールを自動生成
- 編集・コピーして即送信

**企業探し3分 + メール生成2分**で営業の工数を60%削減できました。この本では実際に動いているツールの全実装を解説します。

## 完成イメージ

```
[検索フォーム]

従業員数: 50〜500名

         ↓
[検索結果テーブル]
△△システム株式会社 | 150名 | 東京都渋谷区

```

## 事前準備

### gBizINFO APIキーの取得

gBizINFOは経済産業省が提供する企業情報データベースです。無料で利用できますが事前登録が必要です。

1. https://info.gbiz.go.jp/ にアクセス
2. 「API利用申請」から登録
3. メールでAPIキーが届く（通常1〜3営業日）

### Anthropic APIキーの取得

https://console.anthropic.com/ でAPIキーを発行してください。
